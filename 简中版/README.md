## 📖 简介

操作哲学是一本探讨“有限观察者”如何在物理宇宙的熵增洪流中求存的生存工程学著作。

本书源于作者（唐乐）三十年的深度反思。它试图为所有想要“活着”的系统——从个人、公司到文明——构建一套可用于分析、预警和避免“想得美，死得快”的理论框架与实操工具。

本书不是圣经，是工具箱。它不提供新的“真理”或“主义”，不教你“应该怎么活”，只提供一套防忽悠、防作死的分析工具，帮你识别认知陷阱，看清“这么干，会不会死”。

---

## 🧠 这是一本怎样的书?

本书的研究对象，是所有以“求存”为元价值的M-I协同流，横跨：个人、团队、组织、政府、文明，乃至未来的硅基智能。

当你的“好心”办了“坏事”，当你依据某个“伟大理念”行动却撞得头破血流，本书尝试回答：

1.  为什么会发生“好心办坏事”？ —— 因为你的理念在轨道一（逻辑纯推演）里跑得通，但在轨道三（物理宇宙）的铁律前，是张空头支票。
2.  造成这种悲剧的根源是什么？ —— 是“释经人”把应然的漂亮话，当成了实然的命令，然后用它来发放“吃人许可证”。
3.  我该如何思考、决策和行动，才能避免被现实摁死？ —— 用“三轨分析法”提前验伤，用“穷举-校验”法筛选方案，当一名清醒的“赌棍”。
